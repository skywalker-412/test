# Web UI for Smart Event Booking Platform

This is a simple Next.js dashboard to access and check the status of all microservices.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.

You will see links to all running microservices. Click any to open its endpoint.
